Installation
=======
1. Install Node.js
1. Run `npm install -g bower` and `npm install -g nws` (with root privileges if needed)
1. Go to the application's directory
1. Run `bower install`
1. Run `nws`
1. Open in browser [localhost:3000](http://localhost:3030)
